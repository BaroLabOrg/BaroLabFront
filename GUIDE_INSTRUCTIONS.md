# Инструкция по созданию руководств (Markdown)

Руководства в BaroLab поддерживают стандартный синтаксис **Markdown**, а также имеют несколько кастомных стилистических элементов (например, красивые цитаты и инфоблоки).

Ниже приведены примеры того, как писать код в левом окне редактора, и как он будет выглядеть в реальности.

---

## 1. Базовое форматирование текста

Вы можете использовать выделение слов.

**Напишите этот код в редакторе:**
```markdown
Здесь есть **жирный текст**, немного *курсива* и ~~зачеркнутых слов~~.
```

В результате на странице отобразится:
Здесь есть **жирный текст**, немного *курсива* и ~~зачеркнутых слов~~.

---

## 2. Заголовки

Используйте символ `#` (решетка) перед текстом для создания заголовков разных размеров.

**Напишите этот код в редакторе:**
```markdown
# Огромный заголовок
## Большой заголовок
### Подзаголовок
#### Маленький подзаголовок
```

В результате на странице отобразится:
# Огромный заголовок
## Большой заголовок
### Подзаголовок
#### Маленький подзаголовок

---

## 3. Списки

Списки можно делать как маркерами, так и цифрами.

**Напишите этот код в редакторе:**
```markdown
- Батискаф "Дюгонь"
- Подлодка "Орка"
  - Особая версия (с рельсотроном)

1. Надеть скафандр
2. Взять сварку
3. Выйти в шлюз
```

В результате на странице отобразится:
- Батискаф "Дюгонь"
- Подлодка "Орка"
  - Особая версия (с рельсотроном)

1. Надеть скафандр
2. Взять сварку
3. Выйти в шлюз

## 5. Кастомные элементы (Специально для BaroLab)

Мы сделали специальные графические стили для двух элементов: **Цитат** и **Инфоблоков**.

### 🔵 Цитаты персонажей (Стилизованные)

Вместо обычных цитат мы сделали красивый синий блок с линией. Чтобы указать автора, в конце поставьте длинное тире ` — ` и имя.

**Напишите этот код в редакторе:**
```markdown
> "Плоть слаба, но кровь спасет меня." — Мусорщик
```

В результате на странице отобразится:
> "Плоть слаба, но кровь спасет меня." — Мусорщик
*(На сайте это превратится в специальный голубой прямоугольник, где текст будет курсивом, а имя "Мусорщик" уйдет вправо и станет серым)*

### 🗂️ Инфоблок (Таблица справа)

Если вам нужно сделать карточку предмета/моба, которая "прилипнет" к правому краю, а обычный текст будет её плавно обтекать слева. Для этого самая первая ячейка заголовка таблицы должна начинаться со слова `INFOBOX:`.

**Напишите этот код в редакторе:**
```markdown
| INFOBOX: Scavenger | |
| :--- | :--- |
| **Faction** | Сектанты |
| **Health** | 100 HP |
| **Attack** | Ближний бой |
```

В результате на странице отобразится:
| INFOBOX: Scavenger | |
| :--- | :--- |
| **Faction** | Сектанты |
| **Health** | 100 HP |
| **Attack** | Ближний бой |
*(На сайте эта таблица уменьшится по ширине, улетит вправо, получит темный стильный фон, а текст под ней поднимется вверх и обтечет её слева).*

## 📝 Полный пример 

Попробуйте скопировать весь этот блок в левое окно редактора и вы увидите как строится полноценная страница:

```markdown
# Мусорщик (Scavenger)

| INFOBOX: Основная информация | |
| :--- | :--- |
| **Фракция** | Культ Крови |
| **Уровень угрозы** | Высокий |
| **Оружие** | Ржавая труба |

Мусорщики — это опасные противники, которых можно встретить в заброшенных секторах станции. Они очень агрессивны и передвигаются группами.

> "Я слышал их шаги в вентиляции. Они не ищут еду... они ищут нас." — Инженер Карлсон

## Тактика боя

1. **Держите дистанцию**: В ближнем бою они наносят огромный урон.
2. **Используйте огонь**: Мусорщики боятся света и огня, бросайте фаеры.

Если вы встретили более трех мусорщиков одновременно, лучшим решением будет **бежать**.
```
